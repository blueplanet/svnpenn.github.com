This is DVDauthor for Win32, version 0.7.0.

These binaries were taken from DVDStyler

http://www.dvdstyler.org/en/

I contacted the author Alex Thüring, alex@thuering.biz and Doc Writer Don Brett, dlbrett@zoominternet.net to try to figure out how they compiled DVDauthor for Windows

Source files are at http://sourceforge.net/projects/dvdauthor

	dvdauthor.exe			-- DVDauthor binary
