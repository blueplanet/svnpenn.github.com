Install Jekyll with Cygwin

Install Cygwin packages

  • gcc4-core
  • git
  • make
  • ruby
  • wget

Install Rubygems

wget rubyforge.org/frs/download.php/76032/rubygems-1.8.23.tgz tar xf rubygems* cd rubygems* ruby setup.rb install cd -

Install posix-spawn

gem install rake-compiler git clone git://github.com/rtomayko/posix-spawn.git cd posix-spawn rake gem gem install pkg/posix-spawn-0.3.6 cd -

PATH=/bin gem install jekyll
