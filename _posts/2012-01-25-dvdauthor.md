This is DVDauthor for Win32, version 0.7.0.

These binaries were taken from DVDStyler

http://www.dvdstyler.org/en/

I contacted the author Alex Thüring, alex@thuering.biz and
Doc Writer Don Brett, dlbrett@zoominternet.net
to try to figure out how they compiled DVDauthor for Windows

Source files are at
http://sourceforge.net/projects/dvdauthor

Package contents:
	
	dvdauthor-0.7.0-win32/
		README.txt				-- This file
		dvdauthor.exe			-- DVDauthor binary
		iconv.dll				-- supporting DLL
		zlib1.dll				-- supporting DLL

Documentation
The ever-evolving book is located at
http://dvdauthor.sourceforge.net/doc/index.html

Author DVD
dvdauthor -o VTSBASE --title NTSC.mpg
dvdauthor -o VTSBASE --toc
